# Troubleshooting

## Common Issues

1. **Blank screen**
   - Ensure all dependencies are installed.
   - Run `npm start` and check browser console for errors.

2. **Boot screen stuck**
   - Check if JavaScript is enabled.
   - Verify that `BootScreen.jsx` is correctly imported.

3. **App windows not draggable/resizable**
   - Make sure `react-rnd` and `framer-motion` are installed.
